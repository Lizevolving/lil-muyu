# 电子木鱼Pro 🪵

> 功德无量 · 解压神器 · 专为打工人和学生设计

## 项目简介

电子木鱼Pro是一个现代化的网页解压工具，以佛教木鱼为灵感，结合现代互动设计理念，为压力山大的打工人和学生提供一个轻松有趣的发泄平台。

## 核心功能

### 🎯 基础敲击
- **普通点击**：基础功德+1，配有经典"咚"声音效
- **暴击模式**：10%概率触发，功德大幅提升(50-150)，伴随震撼音效和屏幕闪光
- **连击系统**：快速连续点击触发连击奖励，5连击开始获得额外功德

### 🎨 视觉反馈
- **3D木鱼设计**：采用渐变阴影，营造真实质感
- **粒子特效**：点击时金色粒子飞舞，暴击和连击时效果更加震撼
- **动态背景**：禅意图案缓缓浮动，营造宁静氛围
- **佛光普照**：达到100次敲击后解锁特殊背景效果

### 💬 趣味文字气泡
根据不同触发条件显示对应文字：

**普通模式**：
- "南无阿弥陀佛"、"功德+1"、"心如止水"、"佛光普照"

**暴击模式**：
- "老板暴毙，功德+100"
- "KPI清零，天降福报"  
- "论文自动写完，功德+50"
- "教授突然不点名，功德+20"
- "天劫化解，功德+999"
- "顿悟成佛，直接开摆"

**连击模式**：
- "功德无量，原地飞升"
- "系统检测到您已开悟"
- "连击成佛"、"梵音阵阵，功德连连"

### 🏆 成就系统
- **初心者**：敲击10次
- **修行者**：敲击100次  
- **大师**：敲击1000次
- **连击王**：达成20连击

### 📊 数据统计
- **总功德值**：累计获得的功德点数
- **敲击次数**：总点击次数统计
- **最高连击**：历史最高连击记录

## 技术特性

### 🎵 音效系统
- 使用Web Audio API生成实时音效
- 三种不同频率音效：普通(200Hz)、暴击(100Hz)、连击(400Hz)
- 自动适配浏览器音频策略

### 💾 数据持久化
- 使用localStorage本地存储所有数据
- 支持跨会话数据保持
- 自动保存进度和成就

### 📱 响应式设计
- 完美适配移动设备和桌面端
- 触摸设备优化交互
- 动态字体和布局调整

### ⚡ 性能优化
- 限制同时存在的文字气泡数量（最多3个）
- 使用transform代替position进行动画，GPU加速
- 粒子效果自动清理，避免内存泄漏

## 使用方法

1. **直接点击**：点击中央的木鱼即可开始敲击
2. **键盘操作**：按空格键也可以敲击木鱼
3. **连击技巧**：在500ms内快速连续点击可触发连击
4. **查看成就**：右侧面板实时显示成就进度
5. **数据保存**：所有数据自动保存到浏览器本地
